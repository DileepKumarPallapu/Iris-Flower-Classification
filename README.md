# Iris Flower Classification

## Overview
Iris Flower Classification is a machine learning project that classifies Iris flowers into Setosa, Versicolor, and Virginica species based on sepal and petal measurements. This project demonstrates supervised learning and classification techniques.

## Features
- Classifies Iris flower species
- Data preprocessing
- Data visualization
- Model training
- Prediction and evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Structure
Iris-Flower-Classification/
│── iris_classification.py
│── iris.csv
│── README.md

## How It Works
1. Load Iris dataset.
2. Preprocess data.
3. Visualize patterns in data.
4. Train classification model.
5. Predict flower species.
6. Evaluate model performance.
